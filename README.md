# ClimbingMountain
팀장 - 류태욱 
팀원 - 정현수, 정찬준, 곽승옥, 최혜준

### 전남 지역 명산 정보 제공 서비스 개발 프로젝트
##### 1.  개요
- 산 , 등산로 정보 제공
- 등산자들을 위한 게시판, 본인만 볼 수 있는 다이어리게시판
- 날씨 정보 제공

##### 2.  스킬
- Java (Jsp/Servlet) 
- OracleDB
